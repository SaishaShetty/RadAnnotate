# **RadAnnotate: Synthetic Clinical Report Annotation Pipeline**

**RadAnnotate** is an experimental pipeline that generates and annotates radiology clinical reports using large language models.

## **Overview**
- **Goal**: Build a data-efficient clinical annotation pipeline combining real and synthetic data.
- **Model**: Mixtral-7B-v3 (for generation) and Qwen-2.5 (for fine-tuning).
